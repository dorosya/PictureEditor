# Photo Editor CLI

  

Консольное приложение для редактирования фотографий на C#.  

Позволяет добавлять изображения, применять фильтры, обрезать, вращать, менять пиксели и создавать коллажи. Поддерживает сохранение и загрузку состояния.

  

---

  

## Возможности

  

- Добавление и просмотр фотографий

- Применение фильтров:

  - Яркость

  - Цветовой фильтр (R, G, B)

- Обрезка и поворот изображений

- Изменение отдельных пикселей

- Создание коллажей из нескольких изображений

- Сохранение и загрузка состояния приложения

## Установка и запуск

Склонируйте репозиторий:

```bash
git clone https://github.com/dorosya/PictureEditor
```

Сборка проекта:

```bash
dotnet build
```

Запуск:

```bash
dotnet run --project PhotoEditor.csproj
```

## Доступные команды CLI

|Команда|Описание|
|---|---|
|`add <путь>`|Добавить фото|
|`list`|Показать список фото|
|`open <id>`|Открыть фото в Windows|
|`brightness <id> <factor>`|Изменить яркость (0.5–2.0)|
|`color <id> <r> <g> <b>`|Цветовой фильтр|
|`crop <id> <x> <y> <w> <h>`|Обрезка изображения|
|`rotate <id> <угол>`|Поворот изображения|
|`pixel <id> <x> <y> <r> <g> <b>`|Изменение конкретного пикселя|
|`collage <id1,id2,...> <путь>`|Создать коллаж|
|`save`|Сохранить состояние|
|`clear`|Очистить список фото|
|`help`|Показать справку|
|`exit`|Выход с сохранением|
  
## Примеры использования

Добавление фото:

```markdown
> add C:\Images\photo1.jpg
```

Изменение яркости:

```markdown
> brightness 1 1.5
```

Создание коллажа:

```markdown
> collage 1,2,3 C:\Images\collage.png
```
